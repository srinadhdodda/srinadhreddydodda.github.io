Project Description

Developed a comprehensive data processing framework for the dataset, focusing on subsetting, cleaning, and reformatting data to enhance analytical efficiency. The project entailed creating three targeted DataFrames, each meticulously organized to include specific columns and formatted to the required data types. This process involved rigorous data cleaning to ensure accuracy and consistency. The final DataFrames were integrated into a SQL database. This integration facilitated seamless data retrieval and complex querying, significantly optimizing data organization and paving the way for more efficient and insightful data-driven decision-making.

Key Features

  • Data Subsetting: Splitting the original dataset into three distinct DataFrames.
  • Data Cleaning: Comprehensive cleaning of the dataset to ensure accuracy and consistency.
  • Data Reforming: Reformatting the data to align with the specified data types.
  • SQL Database Integration: Importing the processed DataFrames into a SQL database.

Technologies Used

  • Numpy
  • Pandas
  • PostgreSQL
