Toy Car Robot design in SolidWorks

Toy Car Robot using ROS

  • Project Details: Github

Project Description

The objective was to build a toy robot model in SolidWorks, integrate LIDAR sensors, and simulate its behavior in Gazebo. The robot was designed to perform teleoperation and execute specific tasks using publisher and subscriber nodes in ROS.

Technologies Used

  • SolidWorks for 3D modeling and assembly
  • ROS (Robot Operating System) for teleoperation, sensor integration, and control
  • Gazebo for simulation
  • Rviz for visualization

Achievements

  • Successfully implemented teleoperation and specific tasks, demonstrating the robot's capabilities.
  • Effectively utilized LIDAR sensors for real-time environmental interaction.
