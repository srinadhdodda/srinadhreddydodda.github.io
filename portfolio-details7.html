Project Description

Designed and integrated a specialized robotic gripper with the UR5 arm to enhance its functionality for precise pick-and-place operations. Developed a robust ROS-based communication system for actuation control to ensure seamless operation.

Technologies Used

  • SolidWorks for 3D modeling and assembly of the UR5 robot
  • ROS (Robot Operating System) for robot control and communication
  • Gazebo for simulating the robot's environment and tasks
  • Rviz for visualizing the robot's operations and sensor data
